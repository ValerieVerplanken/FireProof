Welkom bij Fireproof, een webapp speciaal gemaakt om mensen te leren omgaan met brandpreventie thuis. Via deze webapp kunt u verschillende filmpjes bekijken hoe u bepaalde zaken het beste kunt blussen, zien hoe je brandwonden moet verzorgen, je kan testen hoe veilig je huis is, je kan ook jouw eigen kennis testen en je kan zien welke hulpdiensten er rondom u zich bevinden. Indien je dit wenst kan je een account aanmaken, dan kan je contact opnemen met een preventie- adviseur, zodat je feedback krijgt over jouw eigen huis.
